# 🐾 PetVerse

## Project Title
PetVerse - AI Powered Pet Care & Management System

## Team Members
Member 1 - Backend Developer
Member 2 - Frontend Developer

## Objective
Build an AI-powered platform where users can manage pets, identify pet breeds using images, shop for pet products, book vet appointments, and get pet care guidance.

## Features
- User Login & Signup
- Pet Management
- AI Breed Detection
- AI Chat Assistant
- Pet Care Guide
- Pet Shopping
- Vet Booking
- Pet Adoption
- User Profile

## Tech Stack
Frontend: React.js
Backend: FastAPI (Python)
Database: MySQL
AI: Gemini API
